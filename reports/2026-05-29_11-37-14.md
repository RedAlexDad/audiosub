# Отчёт 2026-05-29 11:37 UTC

## What was added
- `subtitle::split_segment()` — функция стала публичной (`pub`) и реэкспортируется из `subtitle::mod.rs`, доступна из `main.rs` и `tui.rs`
- `make cli` — новый Makefile target для запуска без TUI
- `--no-tui` — новый флаг в `cli.rs`, заменяет `--tui` на противоположность

## What was changed
- **Режим по умолчанию переключён на TUI**: `--no-tui` активирует CLI-режим; без флага — TUI (ratatui). Выход из TUI по `q`/`Esc`.
- `tui::TuiApp::new()` принимает `max_duration_ms`, внутри `add_segments()` теперь вызывает `split_segment()` — TUI показывает уже нарезанные куски
- `run_session()` (CLI) логирует сегменты ПОСЛЕ `split_segment()`, а не до — лог показывает реальные отрезки, попадающие в файл
- `audiosub.toml`: `max_duration_ms = 5000` (было 10000)
- Makefile: `run` теперь пишет "TUI mode", добавлена цель `cli` с `--no-tui`
