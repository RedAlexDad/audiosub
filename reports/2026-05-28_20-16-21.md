## Problems encountered

- cargo build падал с unable to find library -lvosk — библиотека в нестандартном пути

## How they were solved

- libvosk.so скопирован в ~/.local/lib/ — каталог без sudo
- .cargo/config.toml: rustflags = ["-L", "~/.local/lib", "-C", "link-args=-Wl,-rpath,..."] — автоматическая линковка и runtime-поиск библиотеки
- Makefile run обёрнут с LD_LIBRARY_PATH для запуска без ручной установки env
