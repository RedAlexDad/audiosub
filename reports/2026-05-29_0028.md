# Отчёт о коммите

## What was added
- Отчёт `reports/2026-05-29_0028.md`

## What was changed
- **Dockerfile**: убраны `--mount=type=cache` (требуют BuildKit, недоступен из-за блокировки Docker Hub в РФ), добавлен `LIBRARY_PATH=/usr/local/lib` для линковки `libvosk.so`
- **docker-compose.yml**: добавлен `build:` с `network: host`, оставлен `image:` как тег
- **Makefile**: `docker-build` переведён на `DOCKER_BUILDKIT=0 docker build --network host`, `docker-run` на `docker compose up` без `--build`

## Problems encountered
- Docker Hub недоступен из РФ (TLS handshake timeout при получении токена на auth.docker.io)
- BuildKit использует собственный DNS/resolver, не решается `--network host`
- Legacy builder работает, но не поддерживает `--mount=type=cache`
- `rust-lld` не находит `libvosk.so` в `/usr/local/lib` без явного `LIBRARY_PATH`

## How they were solved
- Legacy builder (`DOCKER_BUILDKIT=0`) с `--network host` — обход блокировки Docker Hub
- `--mount=type=cache` заменён на обычный `RUN cargo build` (без кэширования registry)
- Нативная сборка (`make build`/`make run`) как основной рабочий процесс
