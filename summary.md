# Workflow

После каждого коммита необходимо заполнить отчёт в `reports/`.

Хук автоматически создаёт файл `reports/<timestamp>.md` с секциями:
- What was added
- What was changed
- Problems encountered
- How they were solved

Нужно отредактировать этот файл: заменить пустые `-` на описание изменений, проблем и решений на русском языке.

Шаблон не удалять. После заполнения — amend последнего коммита, включив отчёт.
