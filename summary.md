# Workflow

Перед каждым коммитом необходимо создать заполненный отчёт в `reports/`.

Структура отчёта:

| Секция | Описание |
|---|---|
| What was added | Новые файлы, фичи |
| What was changed | Изменения в существующем коде |
| Problems encountered | Какие были проблемы/ошибки |
| How they were solved | Как这些问题 были решены |

Отчёт создаётся вручную, добавляется в индекс вместе с изменёнными файлами, и коммитится за один раз. Без amend, без автоматических хуков.
